# Prompt Manager - 项目概述

## 1. 项目简介

**Prompt Manager** 是一款专为 AI 用户设计的桌面端 Prompt 管理工具。它帮助用户高效地创建、组织、编辑和使用 Prompt 模板，支持变量替换、版本历史追踪、标签分类等核心功能。

## 2. 核心价值

- **效率提升**：快速搜索和调用常用 Prompt，支持一键复制
- **模板化管理**：通过变量系统实现 Prompt 的参数化复用
- **版本追溯**：时光机功能记录每次修改，支持版本对比和回滚
- **跨平台同步**：本地 SQLite 存储，可通过 Git/iCloud 同步

## 3. 目标用户

- AI 重度用户（ChatGPT、Claude、Gemini 等）
- 内容创作者
- 开发者
- 产品经理 / 运营人员

## 4. 技术栈概览

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 桌面框架 | Tauri 2.0 | 轻量级跨平台，Rust 后端 |
| 前端框架 | React 18 + TypeScript | 类型安全，组件化开发 |
| 样式方案 | Tailwind CSS | 原子化 CSS，快速开发 |
| 动画库 | Framer Motion | 流畅的交互动画 |
| 状态管理 | Zustand | 轻量级状态管理 |
| 数据存储 | SQLite (via SQLx) | 嵌入式数据库，高性能 |
| 文本对比 | diff-match-patch | Google 算法库，实现版本对比 |

## 6. 设计原则

1. **简洁优先**：界面简洁直观，减少学习成本
2. **性能至上**：启动快、响应快，Tauri + SQLite 保证性能
3. **数据安全**：本地存储，用户数据不上传云端
4. **可扩展性**：模块化设计，便于后续功能扩展
