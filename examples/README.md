# 示例 Prompt

本目录包含一些预设的 Prompt 示例，您可以导入使用。

## 使用方法

1. 打开 Prompt Manager
2. 点击左上角的设置图标（齿轮）
3. 选择"导入数据"
4. 选择 `sample-prompts.json` 文件
5. 导入完成后即可使用

## 包含的示例

### 1. 代码审查助手
**标签**: 编程, 代码审查, 开发

帮助你以资深开发者的角度审查代码，提供质量、性能和安全方面的建议。

**变量**:
- `language`: 编程语言选择
- `clipboard`: 从剪贴板读取代码
- `tone`: 审查态度（严格/友好/建设性）

### 2. 邮件撰写助手
**标签**: 办公, 邮件, 写作

协助撰写各类专业邮件，支持多种场景和语气。

**变量**:
- `email_type`: 邮件类型（商务/求职/感谢信等）
- `recipient`: 收件人
- `subject`: 邮件主题
- `tone`: 语气（正式/友好/简洁）
- `language`: 语言（中文/英文）

### 3. 周报生成器
**标签**: 办公, 周报, 总结

快速生成结构化的工作周报。

**变量**:
- `date`: 自动填充当前日期
- `clipboard`: 从剪贴板读取工作内容
- `style`: 周报风格

### 4. 翻译助手
**标签**: 翻译, 语言, 工具

支持多语言翻译，保持语气和术语准确性。

**变量**:
- `from_lang`: 源语言
- `to_lang`: 目标语言
- `tone`: 语气风格
- `clipboard`: 待翻译内容

### 5. 文章摘要生成
**标签**: 写作, 摘要, 阅读

从长文章中提取关键信息生成摘要。

**变量**:
- `length`: 摘要长度（简短/中等/详细）
- `clipboard`: 文章内容
- `date`, `time`: 自动时间戳

## 自定义示例

您可以基于这些示例创建自己的 Prompt：

1. 复制现有 Prompt
2. 修改内容和变量
3. 添加新的标签
4. 保存使用

## 分享你的 Prompt

如果你创建了优秀的 Prompt，欢迎分享！

1. 选择要分享的 Prompt
2. 点击"设置" → "导出数据"
3. 生成 JSON 文件
4. 通过 Issue 或 PR 提交到项目
